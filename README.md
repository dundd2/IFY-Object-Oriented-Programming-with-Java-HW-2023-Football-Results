# Object-Oriented-Programming-with-Java-HW-2023-Football-Results
This is a website project created for the Computing Project Homework-TokiPonaSpellChecker in March 2023. For more information please check that 2 docx.
The goal was to create a football results application that allows teams to store information about the matches they have played. The application consists of two classes - FootMatch and Team.

FootMatch
The FootMatch class represents an individual football match. It stores information about:

The home team name
The away team name
Home team goals scored
Away team goals scored
It also contains methods to:

Set the match result after the game is played
Print the match result
Accessor methods for the fields
Team
The Team class stores information about an individual team's match results across a season. This includes:

Team name
A list of FootMatch objects representing the matches played
Goals scored for the team
Goals scored against the team
Total points
It contains methods to:

Update the team's statistics after each match
List matches played in various formats (all, home, away, against a specific opponent etc)
List matches won or lost
Running the Code
The TestAss2 class contains sample code to instantiate Team and FootMatch objects and test the application.

Overall this project gave me exposure to core object-oriented programming principles like encapsulation, inheritance, and polymorphism. I learned how to model real-world components as classes and define relationships between them.
